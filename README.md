# dotfiles

## Setup

- Operating System: `arch`
- Window Manager: `dwm, cwm`
- Shell: `bash`
- Terminal Emulator: `st, alacritty`
- File Manager: `fff, pcmanfm, ranger`
- Application Launcher: `dmenu`
- Compositor: `compton`
- Bar: `dwm bar script, lemonbar,  polybar`
- Notification: `dunst`
- Text Editor: `micro, nvim, code`
