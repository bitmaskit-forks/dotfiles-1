# dotfiles

## Setup

- Operating System: `arch`
- Window Manager: `dwm`
- Shell: `bash`
- Display Manager: `lightdm`
- Terminal Emulator: `st, urxvt`
- File Manager: `fff, pcmanfm`
- Application Launcher: `rofi, dmenu`
- Compositor: `compton`
- Bar: `dwm bar script, polybar`
- Notification: `dunst`
- Text Editor: `micro, code`
