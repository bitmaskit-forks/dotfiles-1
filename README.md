## Setup

- Machine: `thinkpad t440p`
- Hostname: `samurai`
- Operating System: `arch`
- Window Manager: `dwm`
- Shell: `zsh`
- Terminal Emulator: `st`
- File Manager: `fff, pcmanfm`
- Application Launcher: `dmenu`
- Compositor: `picom`
- Bar: `dwm bar & status script`
- Notification: `dunst`
- Text Editor: `nvim, nano, code`
